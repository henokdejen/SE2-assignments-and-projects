** This is file sharing app based on peer to peer architectural design. It is developed using java programming language.
It make use of socket programming concept. This app lets user transfer data from one pc to other wirelessly using the pc 
wireless connectivity. 

** What users of this should do:
	1. Since the app doesn't rememer destination folder for received files, users need to give it always the app is started
	3. The users should first be wirelessly connected using some wifi.
	2. One of the two users should create or initialise connection and the other should join the connection
	3. Since Users need to connect using some wifi, the joining user need to know the creator's ip address
